Нормосфера

Программный комплекс «Нормосфера» — это универсальное решение для автоматизации деятельности представительных органов власти всех уровней. Комплекс имеет модульную структуру, что позволяет использовать поэтапный подход к внедрению и развитию системы.

Модули программного комплекса
  • Система управления повестками заседаний
  • Система электронного документооборота с автоматизацией законотворческих процессов
  • Система дистанционного электронного голосования
  • Сервис автоматизации деятельности органов местного самоуправления
  • Конструктор документов – информационная система для работы с версиями документов с использованием технологий искусственного интеллекта

«Нормосфера» включена в Единый реестр российских программ для электронных вычислительных машин и баз данных, реестровая запись №23573 от 12.08.2024

Геоинформационная платформа для создания веб-карт и геопорталов

SkyoriGIS предоставляет следующие инструменты для работы с геоданными на веб-карте
  • Отображение на карте геоданных — объектов, инженерных сетей и пр
  • Поиск и идентификация по карте
  • Навигация по карте
  • Возможность проверки данных по заданным правилам
  • Возможность создавать новые тематические слои в базе геоданных и затем генерировать их на ГИС-сервере
  • Импорт/экспорт геоданных
  • Развитое API для интеграции во внешние веб-приложения корпоративных систем компании

Возможности Геопортала на базе SkyoriGIS

  • единая точка доступа к геоданным
  • пространственный анализ и мониторинг объектов
  • отчеты по выбранным участкам карты
  • интерактивная веб-визуализация
  • организация удобного каталога геоданных
  • поиск и фильтрация
  • настройка прав доступа для разных категорий пользователей

Нейро-экспертная система SkyoriKardio

В основе системы SkyoriKardio лежат нейросетевые технологии и математические методы. Прогнозирование появления и развития заболеваний выполняет искусственная нейронная сеть, обученная на данных обследований 3000+ пациентов.

Нейро-экспертная система SkyoriKardio предназначена для:
  • Прогнозирования развития сердечно-сосудистых заболеваний с учетом множества факторов, включающих генетические и наследственные особенности, условия и образ жизни
  • Поддержки принятия решений при постановке диагнозов
  • Подбора курсов лечения и профилактики заболеваний с учетом индивидуальных особенностей пациента

Решение создано командой опытных специалистов в области медицины и искусственного интеллекта. Система успешно прошла всестороннее тестирование и опытную эксплуатацию в ряде медицинских центров.

Корпоративный ИИ Эксперт «Skyori AI Expert»

Система позволяет осуществлять «умный» поиск по корпоративным данным и находить информацию не только по ключевым словам, но и по смыслу, контексту и взаимосвязям данных. Система реализована на базе использования больших языковых моделей (LLM) и технологии RAG (Retrieval-Augmented Generation).

Основные возможности Skyori AI Expert:
  • Семантический поиск по внутренним источникам данных
  • Формирование кратких и развернутых резюме по найденным документам
  • Выдача ссылок на релевантные материалы
  • Выбор источников данных для поиска
  • Работа через диалоговый интерфейс в формате чата

Использование Skyori AI Expert обеспечивает:

  • Рост производительности сотрудников
  • Возможность постоянного роста и пополнения базы знаний
  • Повышение скорости и качества принимаемых решений

Пример: Экспертная система по региональному законодательству Законодательного Собрания Пермского края – позволяет пользователям получать ответы на вопросы и проводить анализ по законодательству регионов РФ в диалоговом режиме.
